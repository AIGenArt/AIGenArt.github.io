UPLOAD TO GITHUB PAGES

1. Extract this ZIP file.
2. Open the repository: AIGenArt/AIGenArt.github.io
3. Choose Add file > Upload files.
4. Upload every file and folder from this extracted folder.
5. Replace the existing files when prompted.
6. Commit directly to main.
7. Wait 1–3 minutes, then refresh https://aigenart.github.io/ with Ctrl + F5.

Important folders:
- assets
- data
- scripts

The compiled assets/styles.css is included. GitHub Pages can publish it directly without running npm.
